# Magisk on WSA (Weekly Build)

## 介绍

不想麻烦用Actions又或者没有耐心？没问题！我提供了MagiskOnWSA的每周构建，保证随时都有新版本！

## 如何使用？

下载很简单！只需在右下角的Release点击一下，就可以了！就是这么方便！
